# Context: Who This Workspace Works For

This is the most important file you will fill in. Every agent reads it before it does anything.
A model can only work with the world you put in front of it - this file is that world.

Fill in each section below in your own words. You can write in Turkish. Be specific. The more
honest and concrete this is, the better every agent you build will be. You can come back and
update it any time; it is meant to grow with you.

> Tip: Do not aim for perfect. Fill in what you know today. The agents will help you sharpen it.

> Why specific wins (the idea from the context-engineering lesson): write "I sell jewelry" and an
> agent writes generic posts. Write "I sell handmade minimalist silver rings to women 25-35 who
> value quiet luxury" and the same agent writes posts that sound like you. This file is what
> controls that difference.
