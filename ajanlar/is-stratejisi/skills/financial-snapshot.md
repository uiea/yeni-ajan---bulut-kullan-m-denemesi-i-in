# Skill: Build a Financial Snapshot

## Purpose
Create a transparent financial-health draft from member-supplied revenue, cost, cash, and pipeline data.

## When to Use This Skill
- "Finansal durumumu analiz et."
- "Kaç ay dayanırız?"

## Do NOT Use This Skill For
- Reviewing a single expense -> use `cost-review.md`.
- Financial, tax, investment, or accounting advice.

## Process
1. List supplied period, currency, revenue, costs, cash, and recurring versus one-time values.
2. Calculate net cash flow, burn, and runway only when the required inputs exist.
3. Break down revenue and costs using `.agents/hey-strategist/FINANCIAL_XRAY.md`.
4. Label assumptions and missing metrics, including CAC and LTV where data is absent.
5. Provide no more than three prioritized draft actions.

## Quality Bar
- Each number is traceable to a supplied input or shown as an assumption.
- The draft is not presented as professional accounting advice.

## Rules & Feedback
- Do not round away uncertainty.
  Why: strategy decisions depend on the true data range.
  How to apply: show ranges or missing data explicitly.
