# Skill: Optimize a Sales Funnel

## Purpose
Identify the largest loss point in supplied sales-funnel data and draft a measurable improvement.

## When to Use This Skill
- "Satış hunisindeki sızıntıyı bul."
- "Tekliften satışa geçişi nasıl iyileştiririz?"

## Do NOT Use This Skill For
- A detailed sales-operation pipeline report -> use the Sales Operations agent's `pipeline-review.md`.
- Implementing outreach or pricing changes.

## Process
1. Map supplied stage counts and time periods.
2. Calculate stage conversion and identify the biggest loss.
3. Use the diagnostic prompts in `.agents/hey-strategist/SALES_OPTIMIZER.md`.
4. State evidence, likely cause, action draft, expected effect, and measurement method.
5. Flag any claim based on fewer than five observations as emerging.

## Quality Bar
- The analysis names a real supplied bottleneck or states that data is insufficient.
- Each action has one measurable validation method.

## Rules & Feedback
- Separate diagnosis from execution.
  Why: insight does not authorize outreach, pricing, or system changes.
  How to apply: present implementation as an approval-required next step.
