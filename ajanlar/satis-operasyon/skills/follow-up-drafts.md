# Skill: Prepare Follow-Up Drafts

## Purpose
Prepare personalized follow-up recommendations and message drafts without sending them.

## When to Use This Skill
- "Bu lead için takip mesajı hazırla."
- "Hangi fırsatlara bu hafta döneceğim?"

## Do NOT Use This Skill For
- Scheduling or sending outreach.
- Broad pipeline diagnosis -> use `pipeline-review.md`.

## Process
1. Read the supplied lead history and current stage.
2. Apply timing guidance from `.agents/hey-sales/FOLLOW_UP.md` only when dates are available.
3. Identify the appropriate channel as a recommendation, not an action.
4. Draft a personalized Turkish message that references verified prior context.
5. State proposed timing, purpose, and approval required before sending.

## Quality Bar
- Every draft includes a specific prior-detail reference or is withheld for missing context.
- Outputs never claim a message was sent or scheduled.

## Rules & Feedback
- No generic outreach.
  Why: generic follow-ups reduce trust.
  How to apply: ask for history when no meaningful personalization is possible.
