# İlk Ajan Paketi

This is your starter agent workspace. It is the "ready files" from Lesson 4. You do not start
from a blank page - you start from here, and you build your own agent team on top of it.

Follow the lesson video for the step-by-step setup. This file is just the map.

## What is in this folder

```
ders-4_ilk-ajan-paketi/
├── CLAUDE.md          ← The brain. Routing rules + the rules every agent follows. Read this first.
├── context.md         ← Who you are, your goal, your route. Start here, or let ajan-kurucu fill it in with you.
├── ajanlar/
│   └── ajan-kurucu/ ← The one agent you start with. It helps you build your own agents.
├── şablonlar/
│   ├── standart-ajan/             ← The empty template every new agent is copied from.
│   ├── NEW_AGENT_BOOTSTRAP.md      ← Step-by-step guide for building an agent.
│   └── AGENT_CREATION_CHECKLIST.md ← Quality check before an agent goes live.
└── çıktılar/           ← Where your agents save their work. After a build, your first real result lands here.
```

## How to start (the short version)

1. Open this folder in Claude Code (the VS Code / Cursor extension).
2. Open `context.md` and fill in what you already know - even one or two lines is fine. If you would rather just start, leave it and ajan-kurucu will fill it in with you.
3. Start a new chat and type exactly: **"ajan-kurucu ile ilk ajanımı kuralım"** ("let's build my first agent with ajan-kurucu"). Within a few seconds it will start asking you questions.
4. Answer its questions. It will design, build, and check a brand-new agent that is yours. You never have to open the files inside `ajanlar/` yourself - ajan-kurucu reads and writes them for you.

That is it. From there, every new agent you build makes this workspace more useful and more yours.

## The idea in one line

You are not just using AI anymore. You are building your own system - a small team of agents
that work with your files, your goals, and your pace.
