# Skill: Comparison Visual

## Purpose
Create side-by-side or before/after comparison visuals that make differences immediately obvious.

## Inputs
- Two or more items to compare
- Comparison criteria (3-7 dimensions ideal)
- Winner/recommendation (if applicable)
- Context: objective comparison or opinionated take?

## Process
1. Identify comparison dimensions
2. Choose format: side-by-side cards, matrix/table, slider, or animated toggle
3. Build as single-file HTML with clean layout and color coding
4. Add interactive elements (hover, click, toggle)

## Quality Bar
- The difference/winner is obvious within 2 seconds
- Fair representation - no visual tricks to bias
- Works at 1920x1080
- Clean enough to screenshot for thumbnail use
