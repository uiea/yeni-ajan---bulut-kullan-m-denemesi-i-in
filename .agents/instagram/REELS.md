# Skill: Reels

## Purpose
Script and plan Instagram Reels that maximize reach and follower growth. All Reels go through Instagram's Trial Reels feature before publishing to the main account.

## Serves Goals
- **Reach** - Reels get 2.25x more distribution than static posts
- **Growth** - Reels are the primary discovery format on Instagram

## Reel Types

| Type | Duration | Best For |
|------|----------|----------|
| Quick tip | 7-15 sec | Snackable value, trending hooks |
| Tutorial | 30-60 sec | Step-by-step demos, tool walkthroughs |
| Comparison | 30-60 sec | "X vs Y", before/after, with/without |
| Behind the scenes | 15-30 sec | Building in public, process reveals |
| React/commentary | 15-45 sec | Hot takes on news, trend reactions |

## Process

### Step 1: Topic Selection
- Pull from outlier research (what Reel topics are breaking out?)
- Cross-reference with other content research (topics already validated elsewhere)
- Check MEMORY.md for hook patterns that work/don't

### Step 2: Script the Reel

**Structure:**
```
HOOK (0-3 sec) - Pattern interrupt, bold claim, or question
  |
SETUP (3-8 sec) - Context, why this matters
  |
CORE (8-45 sec) - The actual value / demo / insight
  |
PAYOFF (last 3-5 sec) - CTA: save, share, follow, comment
```

**Hook formulas that work:**
- "Stop using [tool] like this..."
- "[Tool] just dropped [feature] and it changes everything"
- "I built [result] in [short time] with [tool]"
- "Most people don't know [tool] can do this..."
- "[Number] tools that actually [specific outcome]"

**Proven hook patterns (adapt to your niche):**
- Show the result first, then explain how
- Ultra-short promise ("X in under Y minutes")
- Comment bait as hook ("Comment '[keyword]' and we'll send you [resource]")
- Gift/giveaway framing

**Comment bait is mandatory on every Reel.** Add to payoff section at minimum. This is a top engagement driver.

### Step 3: Visual Planning
- Shot list or screen recording plan
- Text overlay placements (Instagram crops to 9:16)
- Background music / SFX notes
- Thumbnail frame selection (for grid appearance)

### Step 4: Trial Phase (Instagram Trial Reels feature)
- Use Instagram's native Trial Reels feature (requires professional account, 1K+ followers)
- Trial Reels are shown ONLY to non-followers - will not appear on grid or followers' feed
- Check metrics after 24 hours: views, likes, comments, shares
- If strong -> enable auto-publish or manually publish to main feed
- Test one variable at a time (hook, length, audio, overlay - not all at once)
- Compare Trial Reels only against other Trial Reels, not regular posts
- Only publish to main account after trial validation

### Step 5: Post-publish (first hour is critical)
- Respond to EVERY comment with 5+ words within 60 minutes of posting
- Stay active for 30 minutes after publishing - early engagement is the strongest ranking signal
- First 24-48 hours determine distribution ceiling - do not post and disappear

## Quality Bar
- Hook must grab in first 3 seconds - no intros, no logos, no "hey guys"
- 50% of viewers drop in first 3 seconds - hook must work visually before audio
- One idea per Reel (pass the "one sentence test")
- Must include comment bait CTA - this is non-negotiable
- Caption must include relevant keywords for Instagram's content AI
- Every Reel must go through Trial Reels feature first
- Brand voice: confident, direct, no fluff
- Avoid vague openers - proven to flop
- Never post without a caption that does work - empty or lazy captions waste reach

## Sweet Spots (2026)
- **7-15 seconds:** loops more, inflates play count - best for simple demos/tips
- **15-30 seconds:** enough for value delivery + comment bait CTA
- **Avoid:** 60-90 seconds - retention data shows most viewers leave before the end
- Use trending audio with under 5,000 uses to ride trends before saturation

## Algorithm Signals - Priority Order
1. **DM shares** - strongest signal (content sent to friends = high value)
2. **Saves** - strong authority signal
3. **Shares** - strong reach signal
4. **Comments** - engagement signal
5. **Likes** - weakest signal in 2026
