# Skill: Single Image

## Purpose
Create high-impact single image posts - data visuals, quote cards, announcements, and infographics that drive engagement and establish authority.

## Serves Goals
- **Engagement** - strong single images generate comments and shares
- **Authority** - data visuals and insights position you as a thought leader

## Image Types

| Type | Best For | Example |
|------|----------|---------|
| Data visual | Benchmark results, comparisons | "Tool A vs Tool B: speed test results" |
| Quote card | Hot takes, insights | "AI won't replace developers. Developers using AI will." |
| Announcement | Product updates, milestones | "We just hit 10K followers" |
| Infographic | Process flows, architectures | "How an AI agent pipeline works" |
| Screenshot + commentary | Tool demos, bug reveals | Annotated screenshot with insight |
| Meme / relatable | Community building, shares | Developer humor with relevant twist |

## Process

### Step 1: Concept Selection
- Pull from outlier research or current events
- Match concept to image type above
- Check if this would work better as a carousel (if yes, hand off)

### Step 2: Design Brief

**Composition guidelines:**
- Dark background unless content demands otherwise
- Bold headline text (large enough for mobile)
- Single focal point - don't try to say too much
- Brand accent colors for highlights
- 1:1 aspect ratio for feed (4:5 if portrait composition works better)

### Step 3: Caption & Hashtags
- Hand off to CAPTIONS skill
- Single images rely heavily on caption for engagement
- Caption should add context the image alone can't convey

## Output Format
```markdown
# Single Image: {concept}

## Type
{data visual / quote card / announcement / infographic / screenshot / meme}

## Visual Brief
**Headline text:** {main text on image}
**Subtext:** {secondary text if any}
**Visual elements:** {what to include}

**Color accent:** {which brand color}

## Design Reference
{Link to outlier or inspiration if applicable}

## Caption Direction
{Key points for the CAPTIONS skill}
```

## What Works (general patterns)
- **Personal milestone + story arc** - sharing your journey resonates
- **Nostalgic/personal framing** - origin stories, throwbacks
- Both outperform corporate/event images by 3-10x

## What Doesn't Work (avoid)
- Corporate/event announcements - speaker panels, event invites -> near-zero engagement
- Posts in a language your audience doesn't engage with
- Partnership/B2B promotional posts - audience doesn't care

## Quality Bar
- Must be readable at Instagram thumbnail size
- One clear message per image
- Brand-consistent visual style
- No generic stock imagery - original or screenshot-based
- Text must be concise (max 10 words for headline)
- If the concept is about a corporate event or partnership -> redirect to Stories (not feed)
