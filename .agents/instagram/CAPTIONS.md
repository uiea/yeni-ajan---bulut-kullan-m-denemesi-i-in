# Skill: Captions & Hashtags

## Purpose
Write captions that drive saves and shares, and select hashtags that help Instagram's AI categorize and distribute your content to the right audience.

## Serves Goals
- **Saves** - educational captions with clear value are the #1 save driver
- **Shares** - relatable insights + clear CTA = shares
- **Reach** - proper hashtags + caption SEO = better algorithmic distribution

## Caption Structure

### For Reels
```
HOOK LINE (first line visible before "...more")
|
2-3 lines of context or value expansion
|
CTA (save / share / follow / comment prompt)
|
.
.
.
Hashtags (in a separate block, below dots)
```

### For Carousels
```
HOOK LINE
|
Expand on the carousel's core message (don't repeat slides)
|
Personal take or additional insight not in the slides
|
CTA
|
Hashtags
```

### For Single Images
```
HOOK LINE
|
Full context (images rely on caption for depth)
|
Opinion / insight / lesson
|
CTA + engagement question
|
Hashtags
```

## Hook Line Formulas
- "I tested [X] for [time] - here's what I found..."
- "Unpopular opinion: [bold claim]"
- "Save this before it gets buried."
- "[Tool] just changed [specific thing] and nobody's talking about it."
- "The [X] trick that 90% of [audience] don't know"
- "Stop scrolling if you use [tool/language]"

## CTA Formulas
- "Save this for your next project"
- "Share this with a friend who needs it"
- "Follow @{your-account} for more tips"
- "Drop a [emoji] if you've experienced this"
- "Which one do you use? Comment below"
- "Tag someone who needs to see this"

**Proven comment bait CTAs (use these):**
- "Comment '[keyword]' and we'll send you [resource]" - drives 100-800+ comments
- "[keyword] as a comment" - gift/giveaway frame
- "Which level are you? Comment below" - question frame with comment bait

**Algorithm CTA priority (strongest -> weakest signal):**
1. DM share ("Send this to someone who needs it")
2. Save ("Save this before you need it")
3. Comment ("Drop '[keyword]' below")
4. Like (don't optimize for this alone)

## Hashtag Strategy

### Rules (updated 2026)
- **3-5 hashtags** per post - 2026 best practice; over-tagging no longer helps reach
- Focus on niche authority, not volume
- Mix of tiers:
  - **Small/niche** (10K-100K posts): most important - your niche-specific tags
  - **Mid** (100K-500K posts): 1-2 max
  - **Large** (1M+): avoid - too competitive, no benefit
- Place hashtags below dot separators or in first comment
- City/location tags work - proven on outlier posts
- Rotate sets - don't use the same exact set twice in a row

### Core Hashtag Bank
Maintained in `config/hashtags.txt` - organized by tier and topic.

Build your hashtag bank around:
- Your niche core tags
- Developer/tech community tags
- Local language tags for your audience

## Caption SEO
Instagram's AI reads captions to categorize content. Include:
- Relevant keywords naturally in the first 2 lines
- Specific tool/technology names
- Action words (build, create, automate, test)
- Audience identifiers (developers, founders, builders)

## Quality Bar
- Hook line must work standalone (it's all users see before tapping "more")
- No generic "Check this out!" or "Amazing!" openers
- Every caption must have a specific CTA - never just "Link in bio" (wastes reach)
- Every Reel caption must include comment bait
- Hashtags must be relevant - 3-5 only, no spam tags
- Captions should add value beyond the visual content
- Match brand voice: confident, direct, no fluff
- Write in the language your audience engages with most
