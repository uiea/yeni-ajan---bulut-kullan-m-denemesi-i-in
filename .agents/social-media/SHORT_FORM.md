# Skill: Short-Form Scripts (Reels + TikTok)

## Purpose
Write 2 short-form video scripts (15-60 sec) from a YouTube video - one for the strongest hook, one for the most actionable tip.

## Serves Goals
- **Reach** - Reels and TikTok have the highest organic reach potential
- **Traffic** - "Full video link in bio" drives viewers to YouTube

## Inputs
- Video transcript (full text)
- Clip timestamps from CLIPS skill (which segments to use)
- Top comments (what moments resonated)

## Process
1. Review clip timestamps from CLIPS skill - pick the 2 strongest moments
2. For each clip, write a script:

### Script Structure
```
[HOOK - first 2 seconds, on screen immediately]
The single most surprising/valuable sentence. No intro. No greeting.

[BODY - 10-40 seconds]
The context, the insight, the proof.
Keep sentences short. One idea per sentence.
Use the transcript's natural spoken language, cleaned up for clarity.

[CTA - last 5 seconds]
Soft: "Full video in link" or "Follow for more"
```

3. Script A: **The Hook Clip** - the most surprising/attention-grabbing moment
4. Script B: **The Tip Clip** - the most actionable takeaway
5. For each script, note:
   - Approximate timestamp in original video
   - On-screen text suggestions
   - Whether it needs B-roll, screen recording, or talking head

## Outputs
- 2 scripts in `outputs/short-form/YYYY-MM-DD_video-slug.md`

## Quality Bar
- Hook must work in the first 2 seconds
- No intro, no greeting - jump straight into value
- When read aloud, script should be 15-60 seconds
- Each script must be understandable WITHOUT watching the full YouTube video
- On-screen text suggestions should reinforce the audio, not duplicate it

## Integration
- Depends on CLIPS skill running first (provides timestamps)
- Short-form performance data feeds back into memory
