# Skill: Twitter Threads

## Purpose
Transform a YouTube video into 2 Twitter thread drafts - each standalone, each highlighting a different angle.

## Serves Goals
- **Reach** - threads get algorithmic distribution when the first tweet hooks
- **Traffic** - last tweet links back to the full video
- **Engagement velocity** - first 30 minutes after posting determine total reach

## Inputs
- Video transcript (full text)
- Key content blocks (thesis, surprising stats, personal stories, actionable tips)
- Top comments from the video

## Process
1. Pick 2 different angles from the same video
2. For each angle, write a thread:
   - **Tweet 1 (hook):** Most surprising/valuable thing. No preamble. Must deliver standalone value.
   - **Tweets 2-5 (body):** Supporting points, each standalone and valuable. Use numbered lists, stats, or actionable steps.
   - **Tweet 6-7 (close):** Key takeaway + reply-generating question + link to full video
3. Check each tweet is under 280 characters
4. Check: would this thread still be valuable if the video didn't exist?

## Video Clip Strategy
- NEVER upload full-length YouTube videos (10+ min) to Twitter natively
- Instead, extract a 60-90 second clip of the most impactful moment
- Full video link goes in the last tweet or as a reply

## Quality Bar
- First tweet must make someone stop scrolling
- No tweet is just a transition - every tweet must have value
- Thread uses the audience's actual language, not formal/generic phrasing
- 3-7 tweets total
- NEVER use all-caps hype words - triggers spam perception
- NEVER use overused CTA patterns like "watch this" with pointing emoji
- Last tweet should include a reply-generating question

## Integration
- Uses clip timestamps from CLIPS skill
- Feeds engagement data back into memory during weekly review
