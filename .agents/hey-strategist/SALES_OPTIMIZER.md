# Skill: Sales Optimizer

## Purpose
Find and fix the biggest leaks in the sales funnel. Every lost deal is data. Every won deal is a pattern.

## Analysis Framework

### Step 1: Funnel Snapshot
Map with ACTUAL numbers:
```
Raw Leads -> Contacted -> Qualified (HOT/WARM/COLD) -> First Meeting -> Demo -> Proposal -> Closed Won/Lost
```
**The bottleneck is the stage with the biggest % drop.**

### Step 2: Bottleneck Analysis
Apply first principles at each stage:
- Lead -> Contact: speed to lead, contact method, timing
- Contact -> Qualified: questions too aggressive? wrong segment? PMF issue?
- Qualified -> Meeting: follow-up timing, scheduling friction
- Meeting -> Demo: discovery vs pitch quality, right person present?
- Demo -> Close: pricing, DM involvement, competitor, implementation fear

### Step 3: Win/Loss Analysis
For every won deal: source, time to close, size, industry, key objection overcome, why they bought
For every lost deal: source, stage lost, reason, competitor, price factor, re-engageable?

### Step 4: Pattern Recognition
- Ideal Customer Profile (from wins): industry, size, pain point, timeline, deal size
- Anti-patterns (from losses): red flags, stall stages, unbeatable objections

### Step 5: Recommendations
```
FINDING: [data shows]
SO WHAT: [revenue impact]
DO THIS: [specific action]
EXPECTED IMPACT: [projected improvement]
MEASURE BY: [how we'll know]
```

## Sales Velocity Formula
```
Velocity = (Opportunities x Avg Deal Value x Win Rate) / Sales Cycle Length

To double velocity:
1. Double opportunities (hardest)
2. Double deal value (medium)
3. Double win rate (medium)
4. Halve sales cycle (often easiest)
```
