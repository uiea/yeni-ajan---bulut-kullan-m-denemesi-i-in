# Skill: Cost Killer

## Purpose
Audit every expense. Challenge every cost. Find the minimum viable spend. Every dollar saved extends runway.

## The Five Questions (For Every Cost)

1. **What happens if we stop paying tomorrow?**
   Nothing -> KILL | Inconvenience -> FIND ALTERNATIVE | Revenue drops -> KEEP | Company dies -> ESSENTIAL

2. **Is there a free or cheaper alternative?**
   Open source? Free tier sufficient? Cheaper competitor? Build in <1 day?

3. **Are we using this to full potential?**
   <50% capacity -> Downgrade | <10% -> Probably don't need it

4. **Can we negotiate a better price?**
   Annual discount? Startup discount? Competitor leverage? Volume discount?

5. **Does this directly generate revenue?**
   YES -> Optimize | INDIRECTLY -> Quantify | NO -> Strong cut candidate

## Action Buckets
- **KILL** (save immediately): unused subs, duplicates, "just in case" costs
- **DOWNGRADE** (save within a week): overprovisioned services, premium tiers
- **RENEGOTIATE** (save within a month): annual discounts, startup programs
- **REPLACE** (save within a month): expensive tools with cheaper alternatives
- **KEEP** (but monitor): revenue-generating essentials

## Zero-Based Budget
Start from zero. What's the ABSOLUTE MINIMUM to operate?
```
Minimum Viable Total: [X/month]
Current Actual: [Y/month]
Waste: [Y-X] = [Z]% of spend
```
