# Skill: Survival Dashboard

## Purpose
Weekly snapshot of company health. One page. Five minutes to read. Answers: "Are we going to make it, and what do we do this week?"

## Dashboard Structure

### Section 1: Vital Signs (Traffic Light)
```
METRIC                 VALUE      STATUS     TREND
Runway (months)        [X]        R/Y/G      up/down/flat
Monthly Revenue        [X]        R/Y/G
Monthly Costs          [X]        R/Y/G
Net Cash Flow          [X]        R/Y/G
Active Pipeline Value  [X]        R/Y/G
Close Rate             [X%]       R/Y/G
Leads This Month       [X]        R/Y/G
Customers (active)     [X]        R/Y/G
```

Thresholds: Runway G>12mo Y6-12 R<6 | Close Rate G>15% Y8-15% R<8% | Pipeline G>3x target

### Section 2: This Week's Numbers
Revenue, costs, pipeline deals, meetings scheduled, deals closing, new leads

### Section 3: Graphs
1. Revenue vs Burn (last 12 weeks, dual-axis line)
2. Pipeline Health (horizontal stacked bar: HOT/WARM/COLD)
3. Runway Projection (line with scenario bands: current/best/worst)
4. Weekly Conversion Funnel (vs 4-week average)

### Section 4: Critical Alerts
Auto-generate when: runway <6mo, no deals closed 2+ weeks, pipeline drops >20%, cost spike >20%, close rate <8%, churn detected

### Section 5: Top 3 Priorities This Week
Exactly 3 actions: highest impact, completable this week, measurable outcome.
Revenue-generating > cost-cutting. Never more than 3.

### Section 6: Hypothesis Scoreboard
Active tests status and progress
